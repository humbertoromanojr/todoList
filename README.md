## Instalation

Copiar projeto do repositório
`git clone https://github.com/humbertoromanojr/todoList`

Instalação do node_modules
`npm or yarn install`

Para iniciar o Bundle
`npm or yarn start`

Para rodar a aplicação
`npm or yarn android or ios`
